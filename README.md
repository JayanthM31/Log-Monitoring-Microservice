# Log Monitoring Microservice

## Description
This project is a backend system that monitors and analyzes logs using REST APIs. It simulates a microservice-based architecture and allows users to retrieve logs, errors, and warnings efficiently.

## Features
- REST API for log retrieval
- Error and warning filtering
- Modular backend design
- Microservice-style architecture

## Endpoints
- /logs → Get all logs  
- /errors → Get only error logs  
- /warnings → Get warning logs  

## Tech Stack
- Python  
- Flask  
- REST API
